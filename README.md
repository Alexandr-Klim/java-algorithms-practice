# Java Algorithms Practice

Репозиторий с решениями алгоритмических задач на Java.

## Темы:
- Строки
- Массивы
- HashMap

## Примеры задач:
- Поиск первой цифры в строке
- Поиск самой длинной последовательности цифр
- Перестановка элементов массива
- Two Sum (HashMap)
- Проверка палиндрома

## Сложность:
- В основном O(n) по времени
- O(1) или O(n) по памяти

## Цель:
Подготовка к алгоритмическим контестам и стажировке.
